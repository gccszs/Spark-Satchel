## 🧠 SparkSatchel 灵犀妙计

**Intelligent Skill Retrieval & Recommendation System for Claude Code**

### 🎯 What is SparkSatchel?

A Meta-Skill that helps you find the perfect tool from thousands of skills through:
- 🧠 Semantic understanding of your intent
- ⚖️ Prudent decision-making (auto-recommend or ask based on confidence)
- 📚 Continuous learning from usage history
- 🌏 Bilingual support (Chinese + English, 50+ languages)

### ✨ Key Features

| Feature | Description |
|---------|-------------|
| **Semantic Retrieval** | Vector-based similarity search |
| **Intent Analysis** | Extracts keywords and entities |
| **Confidence Scoring** | 3-tier decision mechanism |
| **Historical Learning** | Optimizes via feedback |
| **Health Checking** | Monitors skill status |
| **Cache Management** | Smart cleanup strategies |

### 🚀 Quick Start

```bash
git clone https://github.com/gccszs/Spark-Satchel.git
cd Spark-Satchel
pip install -r requirements.txt
```
